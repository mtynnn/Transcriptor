# 🩺 vTranscriptor: Inteligencia Artificial para Veterinaria

vTranscriptor es una aplicación de escritorio diseñada para facilitar la transcripción y organización de apuntes médicos en el área de medicina veterinaria. Combina el poder de **Faster-Whisper** para procesamiento local de audio con una interfaz premium y moderna.

## ✨ Características Principales

- **Transcripción Local Ultra-Rápida:** Utiliza IA optimizada (Faster-Whisper) que corre directamente en tu PC (soporta GPU NVIDIA) para máxima privacidad y velocidad.
- **Contexto Inteligente:** Puedes cargar diapositivas (PPTX) o PDFs para que la IA aprenda los términos técnicos y nombres de medicamentos específicos.
- **Formateo Fluido:** Convierte el audio en párrafos legibles con puntuación inteligente basada en las pausas del habla.
- **Organización por Ramos:** Guarda y clasifica tus archivos automáticamente en carpetas según la asignatura o categoría.
- **Integración con Google Drive:** Vincula múltiples cuentas para descargar audios directamente desde la nube o subir tus apuntes listos.
- **Biblioteca Interactiva:** Un calendario visual para navegar por tus transcripciones pasadas de forma intuitiva.
- **Actualizaciones Automáticas:** Instalador One-Click que se mantiene al día automáticamente vía GitHub.

## 🛠️ Requisitos Técnicos

- **Windows 10/11**
- **Hardware:** Funciona en cualquier PC moderno, pero se recomienda una GPU NVIDIA (serie GTX 1060+) para transcripción en tiempo real.

## 🚀 Instalación para Usuarios

1. Ve a la sección de [Releases](https://github.com/mtynnn/Transcriptor/releases).
2. Descarga el archivo `.msi` más reciente.
3. Instala y ¡listo!
